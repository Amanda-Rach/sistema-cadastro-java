# Sistema de Cadastro em Java 

Este é um projeto simples que estou desenvolvendo para praticar os conceitos de Programação Orientada a Objetos.


## Funcionalidades
- Cadastrar pessoa
- Listar pessoas cadastradas
- Buscar pessoa pelo CPF

## Tecnologias utilizadas
- Java
- Scanner
- ArrayList

## Conceitos aplicados
- Classes e objetos
- Encapsulamento
- Pacotes
- Estruturas de repetição
- Estruturas de decisão


## Como executar
```bash
javac cadastro/Main.java
java cadastro.Main
